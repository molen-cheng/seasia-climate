# 东南亚·云南·海南 天气预报项目

## 项目目标
为泰国（6大区域）、云南、海南搭建每日天气预报系统，用于农业/橡胶业务决策参考。

## 最终交付

### 1. HTML 可视化报告
- **文件**: `scripts/weather_report.html`
- 每个城市一张图表（29天）：前14天历史实际 → 红色虚线标记"今日"(预报值) → 后14天未来预报
- 温度折线图（最高温红色/最低温蓝色）+ 降水柱状图（蓝色渐变）
- 今日概览：温度、降水量（含等级标注如"中雨"）、降水概率
- 14天趋势文字总结
- 底部月度降水对比表（过去14天 vs 未来14天）
- 支持参数过滤：`?region=thai` / `?region=china`

### 2. 图表特性
- 温度Y轴：根据每个城市数据自动调整，上下各留15%间距
- 降雨Y轴：<10mm 固定0-10mm，≥10mm 按实际数据取整
- 使用 chartjs-plugin-annotation 画"今日"标记线
- Chart.js 4 + chartjs-plugin-annotation 3

### 4. 技术方案
- **数据源**: Open-Meteo API（免费，无需 key）
  - 历史数据: `archive-api.open-meteo.com`（前14天实际数据）
  - 预报数据: `api.open-meteo.com`（未来15天，用14天）
  - 更新频率: 每天6次（UTC 00/06/12/18），实际可用数据约延迟3-5小时
- **渲染**: Chart.js 4 + chartjs-plugin-annotation 3
- **截图**: Playwright + Chrome headless（本地 channel='chrome'）
- **推送**: 通过 OpenClaw cron 定时执行

### 5. 降雨等级标准（中国气象）
| 等级 | 24小时降水量 |
|------|-------------|
| 小雨 | < 10mm |
| 中雨 | 10~25mm |
| 大雨 | 25~50mm |
| 暴雨 | 50~100mm |
| 大暴雨 | 100~250mm |
| 特大暴雨 | > 250mm |
